# 神装伝説 - しんそうでんせつ -

ブラウザで遊べるダークファンタジーRPG。GitHub Pages対応。

## 遊び方

**PC:** キーボード操作（矢印キー移動、Z/X/C/Vスキル、スペースジャンプ）
**スマホ:** 仮想パッド（タッチ操作）

## ゲーム概要

- **10ワールド × 5ステージ** のウェーブバトル
- **6職業:** ウォーリア、ブレーダー、ホースシールダー、魔法使い、僧侶、アーチャー
- **装備3層システム:** 固有特性 + スロット特性 + アップグレード(+8まで)
- **60種のモンスター** + 10体のワールドボス
- **60種の装備** (武器/盾/頭/体/足/アクセ × 10種 × 5ランク)
- ガチャ、パーティ編成、ショップ、図鑑、セーブ/ロード

## 技術仕様

- **Canvas 960×540** レスポンシブ対応
- **60FPS** requestAnimationFrameループ
- **LocalStorage** セーブ
- **外部依存なし** (Google Fonts以外)

## デプロイ

```
git init
git add -A
git commit -m "initial commit"
git remote add origin <your-repo-url>
git push -u origin main
```

GitHub Settings → Pages → Branch: main → Save
